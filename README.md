# Member's Only
A basic message posting service developed using Node, Express, Passport and EJS.

This project was created as part of [The Odin Project](https://www.theodinproject.com) curriculum.

# Use
## Pre-made Accounts
The following accounts are available for immediate use:
| Account | Password | Status |
| ------- | -------- | ------ |
| testificate | villager | user |
| sudo man | administrator | member, admin |

## Membership Upgrade
Enter the passcode "upgradePLEASE" in the Upgrade page to upgrade your account.
